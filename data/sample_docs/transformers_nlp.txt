A transformer is a neural network architecture designed for sequence data such as text. Before the model runs, each word or token is mapped to a dense vector by an embedding layer, and a positional embedding is added so the model knows the order of the words. The core operation is self-attention, which lets every token look at every other token in the sequence and weigh how relevant each one is when building its own representation. Stacking attention and feed-forward layers lets the model capture context and long-range dependencies between distant words. For text classification, the token representations are pooled into a single vector that a small dense layer maps to class probabilities. Transformers can be trained from scratch on a labeled dataset, or fine-tuned starting from a large pretrained language model when labeled data is limited.
